Eddies Questbuch – so funktioniert’s

Fotos rein → PDF raus → zuhause ausdrucken. Ohne Stress. Ohne „richtig/falsch“.

In 3 Schritten

1) Fotos auswählen
Am besten 10–24 Fotos (Alltag, Familie, Spielsachen, Haustier…).
2) Buch erstellen
Die App macht aus den Fotos Skizzen + Aufgaben.
3) Zuhause drucken
PDF öffnen → Drucker → ggf. An Seite anpassen → lochen/abheften.

Druck-Tipps

Format
Am einfachsten: A4 im Druckdialog.
Skalierung
An Seite anpassen oder 100% (je nach Drucker).
Papier
Normal geht. Premium: 120–160g.
Heften
Lochen + Ordner / Klemmschiene / Heftklammer.

Sprachunterricht (Version)

Idee
Bild → Wörter/Sätze → kleine Aufgaben (je nach Alter/Level).
